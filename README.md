# Portfolio · Carlos Peraza

Portfolio personal en HTML, CSS y JavaScript vanilla. Enfocado en mi perfil de ciberseguridad: experiencia, proyectos defensivos y ofensivos, stack técnico y laboratorio local de IA.

**Live:** [b4te5.github.io](https://b4te5.github.io)

## Stack

- HTML5 + CSS3
- JavaScript vanilla (sin frameworks)
- [Lucide Icons](https://lucide.dev) para iconografía
- [Simple Icons](https://simpleicons.org) para logos de tecnologías
- Soporte i18n ES/EN integrado
- Despliegue en GitHub Pages

## Estructura

```
.
├── index.html          # Single page con todas las secciones
├── assets/
│   ├── projects/       # Logos de proyectos
│   └── screenshots/    # Capturas para el carrusel
├── og-image.svg        # Open Graph preview
└── README.md
```

## Desarrollo local

No requiere build. Para evitar problemas con CORS al cargar recursos externos:

```bash
# Con Python
python -m http.server 8000

# Con Node
npx serve .
```

Y abre `http://localhost:8000`.
